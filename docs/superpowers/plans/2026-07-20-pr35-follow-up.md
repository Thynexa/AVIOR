### Task 2: Ignore only `.DS_Store` in fixture drift checks

**Files:**
- Modify: `tests/testthat/helper-fixtures.R`
- Modify: `tests/testthat/test-fixture-drift.R`

**Interfaces:**
- Produces: `fixture_rel_files(root)` returning a sorted character vector of recursive relative file paths, excluding entries whose basename is exactly `.DS_Store`.
- Consumes: the helper in the vendored-fixture byte comparison.

- [ ] **Step 1: Add a failing regression test and switch the drift test to the helper**

Add:

```r
test_that("fixture file listing ignores only macOS Finder metadata", {
  left <- tempfile("fixture-left-")
  right <- tempfile("fixture-right-")
  dir.create(left)
  dir.create(right)
  writeLines("same", file.path(left, "artifact.yml"))
  writeLines("same", file.path(right, "artifact.yml"))
  writeBin(charToRaw("finder"), file.path(left, ".DS_Store"))

  expect_identical(fixture_rel_files(left), fixture_rel_files(right))

  writeLines("unexpected", file.path(left, ".unexpected"))
  expect_false(identical(fixture_rel_files(left), fixture_rel_files(right)))
})
```

Replace the local `rel_files()` function in the existing drift test with calls to `fixture_rel_files()`.

- [ ] **Step 2: Run the focused test and verify RED**

Run:

```bash

```

Expected: failure because `fixture_rel_files()` is not defined.

- [ ] **Step 3: Implement the minimal helper**

Add to `tests/testthat/helper-fixtures.R`:

```r
fixture_rel_files <- function(root) {
  files <- list.files(root, recursive = TRUE, all.files = TRUE, no.. = TRUE)
  sort(files[basename(files) != ".DS_Store"])
}
```
